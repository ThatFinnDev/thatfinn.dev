SR2E

SR2E, short for SR2Essentials, is a mod for the popular game Slime Rancher 2 by Monomi Park. The mod requires MelonLoader to work. The mod adds lots of things to the main game, like a console, a mod menu, a cheat menu and bunch of other cool things! This mod is not affiliated with Monomi Park.

I am developing this mod with the help of PinkTarr. This mod is available on GitHub and Nexusmods.

AA-Reboot

AA-Reboot is a very simple homebrew for a modded Switch console. It just reboots to payload. I originally made it, because I was to lazy to press "-" when opening the original app. Also, since it starts with "AA-", it's pretty likely to be the first entry, when staring the homebrew. It makes rebooting to payload faster. Instead of the normal atmosphere one, it looks for hekate.bin on the root of your sd card (if you want a custom payload) and if there is non, it reboots into hekate if installed.

AA-Reboot has an extension called AA-Reboot-APPLET which as the name suggest is an applet forwarder. It replaces the Nintendo Switch Online applet on the home screen. In this scenario, if you have only one account (a linked one) on your Switch, and direct game starting is activated, you only need to press home and double tap on the icon to enter hekate.
